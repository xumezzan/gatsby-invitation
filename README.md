# Latifa's Great Birthday — Gatsby Invitation

Одностраничный веб-пригласительный в стиле «Великий Гэтсби» (арт-деко, золото/серебро/чёрный).
Дата события: **12.07.2026**, место: **Miras, Ташкент**.

Чистый HTML/CSS/JS в одном файле. Без сборки, без зависимостей, без фреймворков.

## Быстрый старт

```bash
# вариант 1: просто открыть index.html в браузере

# вариант 2: локальный сервер (нужен для теста музыки на некоторых браузерах)
npx serve .
# или
python3 -m http.server 8000
```

Проверять обязательно в мобильной ширине (~380px) — приглашение рассылается через Telegram.

## Деплой

Netlify Drop: перетащить **папку целиком** (index.html + og-image.png + **atmos.mp4** + music.mp3) на https://app.netlify.com/drop.
`atmos.mp4` — фоновое видео секции «Атмосфера» (обязательно рядом; если его нет, покажется статичный кадр-постер). `music.mp3` опционально.

## Как устроена страница

Интро «Black Velvet × Champagne Gold» (WebGL-бархат: свет проходит по ткани → золотое тиснение LATIFA → OPEN THE INVITATION → имя растворяется в золотую пыль, переход через «дымчатое стекло») → Экран-афиша с очень крупной датой «12 JULY 2026», MIRAS и countdown одной строкой → Атмосфера вечера (полноэкранное фото в дуотоне + фраза) → Dress code как fashion-галерея (образы For Her / For Him) → Финал (тёмный экран с дымом и золотыми бликами, «A little party never killed nobody» · See you · 12.07.2026) → футер. Музыка стартует по клику на OPEN THE INVITATION, кнопка вкл/выкл — справа внизу. Для интро подключены Three.js и GSAP с CDN; без них (или с reduced-motion) интро работает в упрощённом статичном режиме.

## RSVP — встроенная форма → Google Форма

В финале уже встроена форма: гость вводит **имя**, выбирает **«Я буду / Не смогу»** и жмёт «Отправить» — ответ уходит прямо в Google Форму (и копится в её таблице), не покидая страницы. Бэкенда нет, всё в `index.html`.

Осталось привязать её к твоей Google Форме — нужно вписать 5 констант в `<script>` (ищи блок `RSVP → GOOGLE ФОРМА`).

**1. Создай форму.** [forms.google.com](https://forms.google.com) → «Пустая форма». Название любое. Добавь ровно два вопроса:
   - **Имя** — тип «Текст (строка)».
   - **Придёте?** — тип «Один из списка», два варианта, например `Я буду` и `Не смогу`.
   - Во вкладке **«Ответы»** нажми значок таблицы — ответы будут собираться в Google Таблицу.

**2. Достань технические ID полей.** Открой форму → «⋮» (три точки вверху справа) → **«Создать предзаполненную ссылку»**. Заполни поля любыми словами (`ИМЯ`, `ДА`) → внизу **«Получить ссылку»** → «Копировать ссылку». Получишь URL вида:
   ```
   https://docs.google.com/forms/d/e/1FAIpQL…/viewform?usp=pp_url&entry.1234567=ИМЯ&entry.7654321=ДА
   ```
   Из него бери:
   - число после `/d/e/` и до `/viewform` — это ID формы;
   - `entry.1234567` (стоит перед `=ИМЯ`) — поле имени;
   - `entry.7654321` (стоит перед `=ДА`) — поле «Придёте?».

**3. Впиши 5 констант** в `index.html` (блок `RSVP → GOOGLE ФОРМА`):
   ```js
   var RSVP_FORM   = 'https://docs.google.com/forms/d/e/1FAIpQL…/formResponse'; // ← ID + /formResponse
   var RSVP_NAME   = 'entry.1234567';   // поле имени
   var RSVP_ATTEND = 'entry.7654321';   // поле «Придёте?»
   var RSVP_YES    = 'Я буду';          // текст варианта «да» — ТОЧНО как в форме
   var RSVP_NO     = 'Не смогу';        // текст варианта «нет» — ТОЧНО как в форме
   ```
   Важно: `RSVP_FORM` оканчивается на **`/formResponse`** (а не `/viewform`). `RSVP_YES` / `RSVP_NO` должны дословно совпадать с текстами вариантов в Google Форме, иначе ответ не засчитается.

Пока константы не заполнены, форма при отправке покажет ошибку и подсказку в консоли (в никуда данные не уйдут). Тексты формы переведены EN/RU (ключи `rsvp*` в словаре внизу файла). Проверить: заполни, отправь — в Google Таблице появится строка.
