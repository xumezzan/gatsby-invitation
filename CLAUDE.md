# CLAUDE.md — память проекта

## Что это
Одностраничный веб-пригласительный «Latifa's Great Birthday Celebration» в стиле Great Gatsby / арт-деко.
Событие: 12.07.2026, зал Miras, Ташкент. Рассылается гостям ссылкой через Telegram → основной сценарий просмотра — мобильный.

## Главный принцип: один файл
Весь сайт живёт в `index.html`: разметка, стили, JS и обе иллюстрации (base64 PNG).
Деплой — Netlify Drop. **Не выносить CSS/JS/картинки в отдельные файлы без прямой просьбы владельца** — это сломает его workflow «перетащил один файл — сайт работает».

## Команды
- Локально: открыть `index.html` или `npx serve .` / `python3 -m http.server`
- Сборки нет. Зависимостей нет. Линтеров нет. Не добавлять package.json без запроса.

## Порядок блоков в index.html
1. `<head>`: OG-теги (превью для Telegram), Google Fonts, весь CSS в одном `<style>`
2. `#intro` — интро-экран (двери `.door.left/.right` с половинками рамки, `.intro-layer` с канвасом искр и текстом, кнопка `#enterBtn`)
3. `.hero` — карточка-приглашение (канвас `#sparkles`, арт-деко рамка, лучи, веера, текст по дуге)
4. Countdown → Details (When/Where) → Dress code (палитра + 2 карточки с base64 `<img>`) → Финал (цитата) → футер
5. `<audio id="bgm">` + кнопка `#musicToggle` → единственный `<script>`

## Ключевые константы
- `EVENT_DATE = new Date('2026-07-12T18:00:00+05:00')` — начало `<script>`. 18:00 — заглушка, точное время не подтверждено.
- Палитра в `:root`: `--gold #d4af37`, `--gold-soft #f4e3ae`, `--gold-deep #8f6f22`, `--silver #cdd2da`, фон `#0a0a0c`
- Шрифты: Italiana (display), Poiret One (деко-заголовки/кнопки), Great Vibes (рукописные акценты), Jost 300 (текст)

## Критические паттерны — НЕ ЛОМАТЬ

### 1. Script-шрифт + background-clip:text = обрезка глифов
Градиент красится только внутри бокса элемента; выносные элементы Great Vibes (росчерк L, апостроф, хвост f) выходят за бокс и «стираются».
**Правило: у любого элемента с Great Vibes + `.gold-text` обязательны `line-height ≥ 1.3` и `padding` по горизонтали.**
Уже применено к `.hero-name`, `.finale-see`, `.intro-name`. `.hero-great` (Italiana) держать на `line-height ≥ 1.04`.

### 2. Размеры текста в интро — только `min(vw, rem)`
На Android с увеличенным системным шрифтом браузер раздувает `rem`; `clamp()` с rem-минимумом делал текст шире экрана (баг воспроизводился, чинили).
**Правило: в интро никаких rem-минимумов в font-size. Только `min(Xvw, Yrem)`.** Плюс `.intro-content > * { max-width:100% }`.

### 3. Механика дверей интро
- Двери шириной `50.25%` (перекрытие 0.5% — маскирует субпиксельный шов), правая выше по z-index.
- Центральная золотая полоса — это `.door::after` (по 2px на внутренних кромках обеих створок). До клика `opacity:0; scaleY(0)`. Полоса появляется ТОЛЬКО после клика — это осознанное требование владельца.
- Тайминги по клику: 0–0.5s гаснет `.intro-layer` и вырастает полоса → двери едут с задержкой `.75s`, 1.5s → `intro.remove()` через `2700ms`. Менять тайминги согласованно во всех трёх местах (transition-delay дверей, transition полосы, setTimeout в JS).
- Рамка продублирована в обеих дверях (`.frame-holder` шириной 100vw, в правой прижат `right:0`) — при разъезде визуально «раскалывается».

### 4. Base64-картинки
Два `<img src="data:image/png;base64,...">` в карточках дресс-кода (~42 и ~28 KB, палитровые PNG 690×920, паддинг до 3:4 под слот). Исходники — `assets/`. Блоки помечены `<!-- ФОТО-ПРИМЕР 1/2 -->`. При замене на реальные фото просто меняется `src`; `object-fit:cover` в `.photo-slot img` уже настроен.

### 5. Reduced motion
В конце CSS глобальный киллер `*{animation:none; transition:none}` под `prefers-reduced-motion` + отдельный медиа-блок, который выставляет интро финальные состояния (`opacity:1`, `stroke-dashoffset:0`, `#intro.open{display:none}`). Любая новая анимация с `opacity:0` в стартовом состоянии обязана получить переопределение в этом блоке, иначе у части гостей элемент останется невидимым.

### 6. Музыка
`bgm.play()` вызывается только в обработчике клика `#enterBtn` (autoplay-политика браузеров). Файл `music.mp3` кладётся рядом с index.html; если его нет — `catch` глотает ошибку, кнопка звука не показывается, сайт работает. Громкость 0.35. Лицензионные треки не вшивать — только royalty-free.

### 7. Канвасы искр
Две независимые копии одной системы частиц: `#sparkles` (hero) и `#introSparkles` (интро, останавливается через `canvas.isConnected` после удаления интро). При рефакторинге не объединять без теста — у них разная привязка размеров (hero vs window).

## Стиль кода
- JS в стиле ES5 (`var`, `function`), без модулей и транспиляции
- Комментарии в коде — на русском; маркеры для правок владельца: `ФОТО-ПРИМЕР`, `МУЗЫКА`, `Адрес зала`, `EVENT_DATE`
- Тексты на странице — английские (Гэтсби-стилистика), формат даты 12.07.2026
- Мобайл-ферст: тестировать на ширине 380px И с увеличенным системным шрифтом
